Kilo Code — open source AI-ассистент с поддержкой автономного выполнения задач: генерации кода, рефакторинга, отладки и написания тестов.

## Требования

- VS Code или PyCharm (Community / Professional)
- Расширение / плагин Kilo Code
- API-ключ [polza.ai](https://polza.ai/dashboard)

## Установка Kilo Code

1. **Откройте Extensions**
   Нажмите на иконку Extensions в боковой панели или используйте `Ctrl+Shift+X` (Windows/Linux) или `Cmd+Shift+X` (macOS).

2. **Найдите Kilo Code**
   Введите `Kilo Code` в поиске расширений.

   ![Установка Kilo Code из VS Code Marketplace](../images/kilo-code/kilocode-install.png)

3. **Установите расширение**
   Нажмите **Install** на расширении "Kilo Code: AI Coding Agent, Copilot, and Autocomplete".

4. **Откройте Kilo Code**
   После установки нажмите на иконку Kilo Code в боковой панели.

## Настройка подключения к Polza.ai

1. **Откройте настройки Kilo Code**
   Нажмите на иконку шестерёнки в панели Kilo Code — откроется **Kilo Settings**.

2. **Перейдите во вкладку Providers**
   В левом меню настроек выберите **Providers**.

   ![Вкладка Providers в настройках Kilo Code](../images/kilo-code/kilocode-providers.png)

3. **Подключите Custom provider**
   Прокрутите список вниз до раздела **Custom provider** и нажмите **Connect**.

4. **Заполните настройки подключения**

   - **Provider ID:** `polza`
   - **Display name:** `Polza.ai`
   - **Base URL:** `https://polza.ai/api/v1`
   - **API key:** Ваш ключ с [polza.ai/dashboard](https://polza.ai/dashboard)
   - **Models:** Например, `moonshotai/kimi-k2.5`

   <Note>
     После ввода Base URL список моделей подгрузится автоматически. Если этого не произошло — введите Model ID вручную.
   </Note>

   ![Настройки подключения Polza.ai в Kilo Code](../images/kilo-code/kilocode-connection-settings.png)

## Возможности

- **Автономные задачи** — Kilo Code самостоятельно выполняет многошаговые задачи: создаёт файлы, запускает команды в терминале, исправляет ошибки
- **Написание кода** — генерация кода по описанию на естественном языке
- **Рефакторинг** — улучшение и оптимизация существующего кода
- **Отладка** — поиск и исправление ошибок
- **Автоматизация браузера** — взаимодействие с веб-страницами без участия пользователя
- **Inline autocomplete** — подсказки прямо в редакторе во время набора кода

## Установка Kilo Code

1. **Откройте настройки PyCharm**
   Перейдите в **File → Settings** (Windows/Linux) или **PyCharm → Settings** (macOS), либо используйте `Ctrl+Alt+S` / `Cmd+,`.

2. **Перейдите в раздел Plugins**
   В левом меню выберите **Plugins**, затем откройте вкладку **Marketplace**.

3. **Найдите Kilo Code**
   В строке поиска введите `Kilo Code`.

   ![Поиск плагина Kilo Code в Marketplace PyCharm](../images/kilo-code/kilocode-pycharm-marketplace.png)

4. **Установите плагин**
   Нажмите **Install** рядом с плагином "Kilo Code" и дождитесь завершения установки.

5. **Примените изменения**
   Нажмите **OK** и перезапустите PyCharm если потребуется.

## Настройка подключения к Polza.ai

1. **Откройте панель Kilo Code**
   После установки иконка Kilo Code появится на правой боковой панели PyCharm. Нажмите на неё.

2. **Откройте настройки расширения**
   В панели Kilo Code нажмите на иконку настроек (**Settings**).

3. **Создайте Configuration Profile**
   В разделе **Providers** нажмите **+** рядом с полем **Configuration Profile**. Введите название, например `Polza.ai`.

4. **Заполните настройки подключения**

   - **API Provider:** выберите `OpenAI Compatible`
   - **Base URL:** `https://polza.ai/api/v1`
   - **API Key:** Ваш ключ с [polza.ai/dashboard](https://polza.ai/dashboard)
   - **Model:** выберите из списка или введите вручную, например `moonshotai/kimi-k2.5`

   <Note>
     После ввода Base URL список моделей подгрузится автоматически.
   </Note>

   ![Настройки провайдера Polza.ai в Kilo Code для PyCharm](../images/kilo-code/kilocode-pycharm-provider-settings.png)

5. **Сохраните настройки**
   Нажмите **Save**.
