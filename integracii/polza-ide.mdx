Polza IDE — бесплатная десктопная среда разработки на базе Electron, React и Monaco Editor с встроенным ИИ-агентом, поддержкой MCP-серверов, терминалом, Git и расширениями в формате VSIX. Она нативно интегрирована с экосистемой Polza.ai: авторизация через аккаунт Polza, автоматическое создание API-ключа, отображение баланса и стоимости сессии прямо в интерфейсе.

![Polza IDE — общий вид интерфейса](../images/polza-ide/overview.png)

## Установка

### Шаг 1. Скачайте установщик

Перейдите на страницу последнего релиза и скачайте файл для вашей операционной системы:

<a href="https://github.com/polza-ai/polza-ide/releases/latest" target="_blank">
  → Скачать Polza IDE (alpha version)
</a>

- **Windows** — файл `.exe` (инсталлятор)
- **macOS** — файл `.dmg` или `.tar.gz`

### Шаг 2. Установите приложение

Запустите скачанный установщик и следуйте инструкциям на экране. На Windows потребуется подтвердить запрос UAC.

## Подключение к Polza.ai

### Шаг 3. Откройте настройки модели

После запуска IDE нажмите на **название текущей модели** в правом нижнем углу окна.

![Кнопка выбора модели в правом нижнем углу](../images/polza-ide/model-button.png)

### Шаг 4. Авторизуйтесь через Polza.ai

В открывшейся панели нажмите кнопку **«Настройка API»** и пройдите авторизацию в браузере. После успешного входа **новый API-ключ будет создан автоматически** и привязан к вашему аккаунту — никаких ручных действий не требуется.

![Экран авторизации Polza.ai](../images/polza-ide/auth-screen.png)

<Note>
  Ключ создаётся один раз и сохраняется в IDE. При следующем запуске авторизация не потребуется.
</Note>

### Шаг 5. Выберите модель

Снова откройте настройки модели (правый нижний угол). Теперь вы увидите:

- **Баланс** вашего аккаунта Polza.ai
- **Стоимость** текущей сессии
- **Плагины Polza**, подключённые к агенту
- **Список доступных моделей**

Выберите нужную модель из списка.

![Настройки модели с балансом и списком моделей](../images/polza-ide/model-settings.png)

### Шаг 6. Добавьте рабочую папку

Откройте нужную директорию через меню **«Файл → Открыть папку»** или перетащите папку в боковую панель. После этого IDE готова к работе — ИИ-агент видит ваш проект и может работать с файлами.

## Готово к работе

После добавления папки можно отправлять запросы агенту прямо из IDE — он видит структуру проекта, читает файлы и отвечает в контексте вашего кода.

![Запрос к модели и ответ агента в Polza IDE](../images/polza-ide/chat-response.png)

## Решение проблем

<AccordionGroup>
  <Accordion title="IDE не запускается на Windows">
    Убедитесь, что у вас установлен Visual C++ Redistributable. Попробуйте запустить установщик от имени администратора.
  </Accordion>
  <Accordion title="Авторизация не завершается">
    Убедитесь, что браузер не блокирует всплывающие окна и что вы вошли в аккаунт на [polza.ai](https://polza.ai). Попробуйте повторить шаг 3.
  </Accordion>
  <Accordion title="Модели не отображаются после авторизации">
    Закройте панель настроек и откройте её снова. Если список пуст — проверьте баланс аккаунта на [polza.ai/dashboard](https://polza.ai/dashboard).
  </Accordion>
  <Accordion title="Агент не видит файлы проекта">
    Убедитесь, что рабочая папка открыта (шаг 6). Агент работает только в контексте открытого проекта.
  </Accordion>
</AccordionGroup>
