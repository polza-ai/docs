Cline — AI-ассистент для VS Code и Cursor с поддержкой автономного выполнения задач.

## Требования

- VS Code, Cursor или VSCodium
- Расширение Cline
- API-ключ [polza.ai](https://polza.ai/dashboard)

## Установка Cline

<Steps>
  <Step title="Откройте Extensions">
    Нажмите на иконку Extensions в боковой панели или используйте `Ctrl+Shift+X` (Windows/Linux) или `Cmd+Shift+X` (macOS).
  </Step>

  <Step title="Найдите Cline">
    Введите "Cline" в поиске расширений.

    <Frame>
      <img src="https://storage.yandexcloud.net/polza-docs/polza-docs/images/f1672565fd2906e4b3a48a41259486d411f4724c454a199b785d183d7406f142.png" alt="Установка Cline из VS Code Marketplace" />
    </Frame>
  </Step>

  <Step title="Установите расширение">
    Нажмите **Install** на расширении "Cline".
  </Step>

  <Step title="Откройте Cline">
    После установки нажмите на иконку Cline в боковой панели.
  </Step>
</Steps>

## Настройка подключения к Polza.ai

<Steps>
  <Step title="Откройте настройки модели">
    В панели Cline нажмите на выбор модели (под полем ввода).
  </Step>

  <Step title="Выберите провайдера">
    Выберите **OpenAI Compatible** в качестве API провайдера.
  </Step>

  <Step title="Введите настройки подключения">
    Заполните поля:
    - **Base URL:** `https://polza.ai/api/v1`
    - **API Key:** Ваш ключ
    - **Model ID:** Например, `anthropic/claude-opus-4`

    <Frame>
      <img src="https://storage.yandexcloud.net/polza-docs/polza-docs/images/b9eb26c783a66918e1fec92a494a5cd7b1d15d6936e60b8d83a76e8656bc0d2c.png" alt="Настройки провайдера Polza.ai в Cline" />
    </Frame>
  </Step>
</Steps>

## Рекомендуемые модели для кодинга

| Модель | ID | Особенности |
|--------|-----|-------------|
| Claude Opus 4.7 | `anthropic/claude-opus-4.7` | Топовое качество tool use, лучший выбор для сложных агентских задач |
| Claude Sonnet 4.6 | `anthropic/claude-4-6-sonnet` | Оптимальный баланс скорости и качества, основной рабочий вариант |
| GPT-5.5 | `openai/gpt-5.5` | Сильное понимание контекста, хорошо справляется с запутанными кодовыми базами |
| Qwen 3.7 Max | `qwen/qwen3.7-max` | Надёжный tool use, сильная альтернатива Claude по соотношению цена/качество |
| Kimi K2.6 | `moonshotai/kimi-k2.6` | Открытая модель, особенно сильна во фронтенде и TypeScript |
| GLM 5.1 | `z-ai/glm-5.1` | Компактная и быстрая, справляется с типовыми задачами рефакторинга |
| Gemini 3.5 Flash | `google/gemini-3.5-flash` | Очень быстрая, подходит для простых задач и быстрого итерирования |
| Qwen 3.6 Flash | `qwen/qwen3.6-flash` | Самая экономичная в списке, для рутинных правок и небольших задач |

<Warning>
**Важно: для Cline подходят не все модели**

Cline активно использует вызовы инструментов: редактирование файлов, запуск команд, навигацию по проекту и другие действия внутри среды разработки. Для стабильной работы агенту важно, чтобы модель корректно поддерживала формат tool use и последовательно следовала структуре вызовов.

У некоторых моделей, включая YandexGPT и ряд других, поддержка сложных сценариев tool use может быть ограничена. В Cline это иногда проявляется как ошибки выполнения, зависания или нестабильное поведение агента.

При этом в чат-интерфейсах и более простых клиентах такие модели могут работать корректно, поскольку там инструменты либо не используются, либо применяются в более простом формате.

По мере развития моделей и улучшения поддержки tool use совместимость будет расширяться. Для наиболее стабильной работы в Cline рекомендуем использовать модели из таблицы выше.
</Warning>

## Решение проблем

<AccordionGroup>
  <Accordion title="Ошибка аутентификации">
    - Проверьте правильность API-ключа
    - Убедитесь в наличии средств на балансе
    - Проверьте Base URL: `https://polza.ai/api/v1`
  </Accordion>

  <Accordion title="Модель не найдена">
    - Проверьте правильность Model ID
    - Актуальный список: [polza.ai/models](https://polza.ai/models)
  </Accordion>

  <Accordion title="Медленные ответы">
    - Используйте более быстрые модели (Sonnet вместо Opus)
    - Уменьшите размер контекста
    - Проверьте интернет-соединение
  </Accordion>

  <Accordion title="Расширение не появляется">
    - Перезагрузите VS Code
    - Проверьте что расширение включено в Extensions
    - Откройте через Command Palette: "Cline: Open In New Tab"
  </Accordion>
</AccordionGroup>
