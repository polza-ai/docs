N8N — это визуальная платформа для автоматизации рабочих процессов. С помощью интеграции Polza.ai вы можете использовать сотни AI-моделей прямо в ваших N8N workflows через стандартный OpenAI-совместимый интерфейс.

## Требования

- Установленный N8N (локально или в облаке)
- API-ключ Polza.ai из [polza.ai/dashboard](https://polza.ai/dashboard)
- Базовое знание работы с N8N

## Настройка

<Steps>
  <Step title="Добавьте OpenAI Chat Model">
    В N8N найдите и добавьте ноду "OpenAI Chat Model" в ваш workflow. Для этого введите "openai" в поиске при добавлении ноды.

    <Frame>
      <img src="https://storage.yandexcloud.net/polza-docs/polza-docs/images/9303c925e8aec285377317ccfcca742971aa9d91ef497ded546ad8db320f728a.png" alt="Поиск OpenAI Chat Model в N8N" />
    </Frame>
  </Step>

  <Step title="Создайте credentials">
    Нажмите на создание новых credentials и заполните:
    - **API Key:** Ваш API-ключ Polza.ai
    - **Base URL:** `https://polza.ai/api/v1`

    <Frame>
      <img src="https://storage.yandexcloud.net/polza-docs/polza-docs/images/ddfd22b1ee6df0c73cd577fedddb408f1142ff26c91d773c3e48f598aed08b91.png" alt="Настройка credentials для Polza.ai" />
    </Frame>
  </Step>

  <Step title="Выберите модель">
    В поле Model введите ID модели, например:
    - `deepseek/deepseek-r1` — для reasoning задач
    - `openai/gpt-4o` — универсальная модель
    - `anthropic/claude-3.5-sonnet` — для кодинга
    - `google/gemini-2.5-pro-preview` — мультимодальная

    <Frame>
      <img src="https://storage.yandexcloud.net/polza-docs/polza-docs/images/0d970037cf627f54e305818107645d77d81dfa9305678a154f6ac4151b3666e0.png" alt="Выбор модели в настройках" />
    </Frame>
  </Step>

  <Step title="Готово!">
    Теперь вы можете использовать AI-модели Polza.ai в ваших N8N workflows.

    <Frame>
      <img src="https://storage.yandexcloud.net/polza-docs/polza-docs/images/5593a925c0adb1789628ffa68d0ae2540d5412a961996bc9116ab2d81d271eb2.png" alt="Рабочий workflow с чатом" />
    </Frame>
  </Step>
</Steps>

<Note>
Polza.ai поддерживает **Responses API** в beta-режиме. Если при использовании возникают проблемы, попробуйте отключить опцию "Use Responses API" в настройках модели.
</Note>

## Расширенные настройки

В настройках AI Agent доступны параметры:

| Параметр | Описание |
|----------|----------|
| Temperature | Креативность ответов (0-2) |
| Max Tokens | Максимальная длина ответа |
| Top P | Nucleus sampling |
| Frequency Penalty | Штраф за повторения |
| Presence Penalty | Штраф за повторные токены |

## Решение проблем

<AccordionGroup>
  <Accordion title="Ошибка аутентификации">
    - Проверьте правильность API-ключа
    - Убедитесь, что Base URL указан как `https://polza.ai/api/v1`
    - Проверьте баланс
  </Accordion>

  <Accordion title="Модель не найдена">
    - Проверьте правильность написания ID модели
    - Актуальный список моделей: [polza.ai/models](https://polza.ai/models)
  </Accordion>

  <Accordion title="Превышен лимит токенов">
    - Уменьшите значение Max Tokens
    - Сократите входной текст
  </Accordion>

  <Accordion title="Проблемы с Responses API">
    - Отключите опцию "Use Responses API" в настройках модели
    - Responses API находится в beta-режиме и может работать нестабильно с некоторыми моделями
  </Accordion>
</AccordionGroup>
