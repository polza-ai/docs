## Начало работы

<AccordionGroup>
  <Accordion title="Что такое Polza.ai?">
    Это удобный сервис с единым API для доступа ко всем популярным LLM-моделям. Мы агрегируем модели от OpenAI, Anthropic, Google, Meta* и десятков других провайдеров.
  </Accordion>

  <Accordion title="Как начать пользоваться Polza.ai?">
    1. Создайте аккаунт на [polza.ai/dashboard](https://polza.ai/dashboard)
    2. Пополните баланс в личном кабинете
    3. Создайте API-ключ
    4. Используйте наш API с любым OpenAI-совместимым SDK
  </Accordion>

  <Accordion title="Как устроено ценообразование?">
    Цена указывается за миллион токенов и различается для входящих и исходящих токенов. Некоторые модели берут плату за запросы, изображения или токены рассуждений. История использования доступна в Консоли
  </Accordion>
</AccordionGroup>

## Модели и провайдеры

<AccordionGroup>
  <Accordion title="Какие модели доступны на Polza.ai?">
    Полный список из 400+ моделей доступен в [каталоге](https://polza.ai/models) или через [API моделей](/api-reference/models/list). Мы постоянно добавляем новые модели.
  </Accordion>

  <Accordion title="Какие SDK поддерживает Polza.ai?">
    Polza.ai полностью совместим с OpenAI API. Поэтому любые SDK, которые работают с OpenAI, будут работать и с нами — включая официальные библиотеки для Python, Node.js, Go и других языков.
  </Accordion>
</AccordionGroup>

## Технические вопросы

<AccordionGroup>
  <Accordion title="Какие конечные точки API поддерживает Polza.ai?">
    Мы реализуем спецификацию OpenAI для `/chat/completions`, а также дополнительные эндпоинты:
    - `/api/v1/images/generations` — генерация изображений
    - `/api/v1/videos/generations` — генерация видео
    - `/api/v1/audio/transcriptions` — распознавание речи
    - `/api/v1/audio/speech` — синтез речи
    - `/api/v1/embeddings` — эмбеддинги
    - `/api/v1/models` — список моделей
    - `/api/v1/balance` — баланс аккаунта
  </Accordion>

  <Accordion title="Поддерживает ли API изображения и другие файлы?">
    Да, поддерживаются текст, изображения и PDF в формате URL или base64 для мультимодальных моделей.
  </Accordion>

  <Accordion title="Как работает потоковая передача?">
    Используется Server-Sent Events (SSE). Добавьте параметр `stream: true` в запрос для получения ответа по частям.
  </Accordion>
</AccordionGroup>

## Конфиденциальность и данные

<AccordionGroup>
  <Accordion title="Какие данные сохраняет Polza.ai?">
    Мы сохраняем только базовые метаданные запросов: время, используемая модель, количество токенов. Содержимое запросов и ответов не сохраняется.
  </Accordion>

  <Accordion title="Как Polza.ai обрабатывает данные провайдерам?">
    Мы передаём запросы провайдерам и работаем с ними, чтобы гарантировать неиспользование данных для обучения моделей. Подробнее в разделе [Конфиденциальность](/osobennosti/privacy).
  </Accordion>
</AccordionGroup>

## Оплата и баланс

<AccordionGroup>
  <Accordion title="Как устроена система оплаты?">
    Баланс пополняется в рублях. Возможно ручное пополнение или автоматическое при падении баланса ниже установленного порога.
  </Accordion>

  <Accordion title="Какие способы оплаты приняты?">
    Принимаются банковские карты, СБП и оплата на расчётный счёт для бизнеса.
  </Accordion>

  <Accordion title="Какова политика возврата?">
    Возврат неиспользованных средств возможен в течение 24 часов после оплаты. Обратитесь в [службу поддержки](/glavnoe/support).
  </Accordion>

  <Accordion title="Как отслеживать расходы?">
    Используйте страницу [polza.ai/dashboard](https://polza.ai/dashboard) или [API баланса](/api-reference/other/balance) для актуальной информации.
  </Accordion>

  <Accordion title="Предлагает ли Polza.ai скидки за объём?">
    Стандартных скидок нет, но возможны индивидуальные условия для крупных клиентов. Свяжитесь с нами для обсуждения.
  </Accordion>
</AccordionGroup>
