Nano Banana 2 — модель генерации изображений с теми же возможностями, что и Nano Banana Pro: до 4K разрешения, точный рендеринг текста и до 8 референс-изображений. ID модели: `google/gemini-3.1-flash-image-preview`.

<Note>
Полная документация API: [Генерация медиа](/api-reference/media/create)
</Note>

## Обзор

| Характеристика | Nano Banana | Nano Banana 2 |
|----------------|-------------|----------------|
| ID модели | `google/gemini-2.5-flash-image` | `google/gemini-3.1-flash-image-preview` |
| Max prompt | 20 000 символов | 20 000 символов |
| Разрешения | - | 1K, 2K, 4K |
| Aspect ratio | 10 вариантов | 11 вариантов |
| Референс-изображения | до 8 | до 8 |

### Генерация изображения в высоком разрешении

<CodeGroup>
```typescript TypeScript
const response = await fetch('https://polza.ai/api/v1/media', {
```

## Параметры

| Параметр | Тип | Обязательный | Описание |
|----------|-----|--------------|----------|
| `model` | string | Да | ID модели: `google/gemini-3.1-flash-image-preview` |
| `prompt` | string | Да | Текстовое описание желаемого изображения (до 20 000 символов) |
| `aspect_ratio` | string | Нет | Соотношение сторон: `1:1`, `2:3`, `3:2`, `3:4`, `4:3`, `4:5`, `5:4`, `9:16`, `16:9`, `21:9` |
| `image_resolution` | string | Нет | Разрешение: `1K`, `2K`, `4K` |
| `output_format` | string | Нет | Формат вывода: `png`, `jpeg` |
| `images` | array | Нет | Массив референс-изображений для редактирования (до 8 шт.) |

## Пример запроса (cURL)

```bash
curl -X POST "https://polza.ai/api/v1/media" \
  -H "Authorization: Bearer <POLZA_AI_API_KEY>" \

    "model": "google/gemini-3.1-flash-image-preview",
    "input": {
      "prompt": "Описание изображения",
      "aspect_ratio": "1:1",
      "image_resolution": "2K",
      "images": []
    },
    "async": true
  }'
```
