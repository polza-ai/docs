Модель генерации видео Sora 2. Длительность 10 или 15 сек, до 1 референс-изображения.

<Note>
Полная документация API: [Генерация медиа](/api-reference/media/create)
</Note>

## Обзор

| Характеристика | Значение |
|----------------|----------|
| ID модели | `openai/sora-2` |
| Max prompt | 10 000 символов |
| aspect_ratio | 9:16, 16:9 |
| duration | 10, 15 сек. По умолчанию 10 |
| images | 0–1 |

## Параметры

| Параметр | Обязательный | Описание |
|----------|---------------|----------|
| `model` | Да | `openai/sora-2` |
| `prompt` | Да | Текст, до 10000 символов |
| `aspect_ratio` | Нет | 9:16, 16:9 |
| `duration` | Нет | 10, 15. По умолчанию 10 |
| `images` | Нет | 0–1 |

## Пример запроса (cURL)

```bash
curl -X POST "https://polza.ai/api/v1/media" \
  -H "Authorization: Bearer <POLZA_AI_API_KEY>" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "openai/sora-2",
    "input": {
      "prompt": "Описание видео",
      "aspect_ratio": "16:9",
      "duration": "10",
      "images": []
    },
    "async": true
  }'
```
