Модели генерации видео от ByteDance нового поколения. Поддержка text-to-video и image-to-video, разрешения 480p–720p, длительность до 15 сек, multi-shot сцены и опциональная генерация звука.

**Seedance 2 Fast** — ускоренная версия с аналогичными возможностями. Используйте её, когда скорость важнее максимального качества.

Полная документация API: [Генерация медиа](https://polza.ai/docs/api-reference/media/create)

## Обзор

| Характеристика | Значение |
| --- | --- |
| aspect\_ratio | 16:9, 4:3, 1:1, 3:4, 9:16, 21:9 |
| resolution | 480p, 720p |
| duration | до 15 сек |
| videos | 0–1 (референс) |
| images | 0–5 (референс) |
| generate\_audio | true / false |
| multi\_shots | true / false |

## Параметры

| Параметр | Обязательный | Описание |
| --- | --- | --- |
| `model` | Да | `bytedance/seedance-2` или `bytedance/seedance-2-fast` |
| `prompt` | Да | Описание видео (3–20000 символов) |
| `aspect_ratio` | Нет | 16:9, 4:3, 1:1, 3:4, 9:16, 21:9 |
| `resolution` | Нет | 480p, 720p |
| `duration` | Нет | Длительность в секундах (до 15) |
| `videos` | Нет | 0–1 (видео-референс) |
| `images` | Нет | 0–5 (изображения-референс) |
| `generate_audio` | Нет | Генерация звука. |
| `multi_shots` | Нет | Несколько сцен в одном видео. По умолчанию false |
